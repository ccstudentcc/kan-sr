# Master Index of Project Specifications

This directory is the engineering and experiment specification hub for `KAN-Symbolic_Regression`, focused on reproducibility, maintainability, and auditability.

| Module | Index File | Focus |

| Thinking Guides | `guides/index.md` | Cross-layer design and reuse decisions |

| Language and Comment Policy | `language-policy.md` | English code/spec, English outputs, Chinese notebook markdown |

| DoD Checklist | `checklists/dod-checklist.md` | Definition of Done and pre-commit checks |

## Relationship to Main Repository Documents
- `README.md`: external-facing project introduction and experiment showcase
- `CURRENT_WORK_REFERENCE.md`: current status and improvement roadmap
- `EXPERIMENT_PROTOCOL.md`: unified experiment protocol body
- `spec/`: executable specifications derived from protocol and practice
