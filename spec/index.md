# 项目规范总索引

本目录是 `KAN-Symbolic_Regression` 的工程与实验规范中心，面向“可复现实验 + 可维护代码 + 可审计结果”。

## 模块导航
| 模块 | 索引文件 | 关注点 |
|---|---|---|
| 实验规范 | `experiment/index.md` | 协议、配置、Notebook、结果与质量门禁 |
| 核心代码规范 | `core/index.md` | `efficient_kan` 与脚本层实现约束 |
| 思维指南 | `guides/index.md` | 跨层设计与复用决策 |
| 仓库结构规范 | `repository-layout.md` | 目录分层、命名规则与迁移策略 |
| 语言与注释规范 | `language-policy.md` | 代码英文、输出英文、Notebook markdown 中文 |
| 流程规范 | `workflow.md` | 从需求到提交的标准执行流程 |
| 变更管理 | `change-management.md` | 规范状态与变更提交流程 |
| DoD 检查清单 | `checklists/dod-checklist.md` | 开发完成定义与提交前检查 |

## 与仓库主文档的关系
- `README.md`：对外介绍与实验展示
- `CURRENT_WORK_REFERENCE.md`：阶段性现状与改进方向
- `EXPERIMENT_PROTOCOL.md`：统一实验协议正文
- `spec/`：将协议与实践落到可执行规范
